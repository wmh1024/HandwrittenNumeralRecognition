## 上手指南

克隆此项目

```sh
git clone https://github.com/wmh1024/HandwrittenNumeralRecognition.git
```

基于MNIST训练数据集版本

```sh
python DataByMNIST.py
```

基于自制数据集版本

```sh
python DataByMyData.py
```

## 使用到的框架

- pytorch
- matplotlib
- numpy
